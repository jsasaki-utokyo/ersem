# Carbonate Engine Test Harness

This directory contains a verification harness for the `carbonate_engine` module,
which provides a PyCO2SYS-compatible carbonate chemistry solver for ERSEM.

## Overview

The carbonate-engine solver (`src/carbonate_engine.F90`) computes pH and fCO2 from
DIC (dissolved inorganic carbon) and TA (total alkalinity) using robust root-finding
(Brent's method). This test harness validates the solver against reference values
generated by PyCO2SYS.

**The test driver calls the actual `carbonate_engine_solve` subroutine** (not a
reimplementation), using a minimal mock of `fabm_types` for standalone compilation.

## Files

- `test_driver.f90` - Fortran test driver that calls `carbonate_engine_solve` directly
- `fabm_types_mock.F90` - Minimal mock providing `rk` kind parameter
- `reference_cases.csv` - Reference test cases with 4 pH scales + fCO2
- `generate_reference.py` - Python script to regenerate reference cases using PyCO2SYS
- `Makefile` - Build and run tests
- `README.md` - This file

## Running the Tests

### Prerequisites

1. A Fortran compiler (gfortran recommended)

### Compile and Run

```bash
cd tests/carbonate_engine
make test
```

Or manually:
```bash
gfortran -O2 -o test_driver \
    fabm_types_mock.F90 ../../src/carbonate_engine.F90 test_driver.f90
./test_driver
```

### Test Coverage

The test driver validates eight categories:

1. **4-scale CSV comparison**: 22 cases × 4 pH scales (Total, SWS, Free, NBS) against PyCO2SYS reference CSV
2. **Cross-scale consistency**: 5 cases verifying pH ordering (NBS > Free > Total > SWS) and pCO2 invariance across scales
3. **legacy_mode=.true.**: 5 cases checking backward compatibility with old CO2DYN solver
4. **Sub-zero temperature**: 2 cases for polar waters (-1.5°C) and legacy T-clamping to 0°C
5. **Millero 2010 (opt_k_carbonic=14)**: 5 cases against PyCO2SYS reference with alternative K1K2 formulation
6. **Pressure correction (Pbar > 0)**: 5 cases at 10–500 bar against PyCO2SYS reference
7. **convert_pH_scale round-trip**: 4 cases verifying pH scale conversion accuracy
8. **engine=0 phscale x opt_pH_scale**: 4 cases testing carbonate.F90 pH_total/pH_selected logic for consistent and inconsistent parameter combinations

### Exit Codes

- `0` - All tests passed
- `1` - One or more tests failed, or no test cases found

## Regenerating Reference Cases

The reference cases are generated using PyCO2SYS. To regenerate:

```bash
pip install PyCO2SYS
python generate_reference.py
```

This will overwrite `reference_cases.csv` with fresh values from PyCO2SYS.

## Reference CSV Format

9-column CSV: `T(°C), S(PSU), DIC(mol/kg), TA(mol/kg), pH_total, pH_sws, pH_free, pH_nbs, fCO2(atm)`

Note: fCO2 (fugacity of CO2) is used instead of pCO2 because carbonate_engine
computes fCO2 = CO2*/K0 without a separate fugacity correction factor.

## PyCO2SYS Settings

The reference cases use the following PyCO2SYS settings to match the carbonate_engine:

| Setting | Value | Description |
|---------|-------|-------------|
| opt_k_carbonic | 10 | Lueker et al. (2000), total pH scale |
| opt_k_bisulfate | 1 | Dickson (1990) |
| opt_k_fluoride | 2 | Perez & Fraga (1987) |
| opt_total_borate | 2 | Lee et al. (2010) |
| opt_pH_scale | 1 | Total scale |

### Supported K1K2 Formulations

| ERSEM opt_k_carbonic | PyCO2SYS # | Reference | Native scale |
|----------------------|------------|-----------|-------------|
| 10 (default) | 10 | Lueker et al. (2000) | Total |
| 14 | 14 | Millero (2010) | SWS |

## Tolerances

Tests pass if the calculated values are within:

- **pH**: ±1e-4 (all scales, all formulations)
- **fCO2**: ±1e-6 atm
